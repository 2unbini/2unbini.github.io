# 구조체

```c
typedef struct		s_list
{
	void			*content;
	struct s_list	*next;
}					t_list;
```

문제에서 주어진 링크드 리스트의 기본적인 구조체이다.

구조화된 데이터를 처리할 때 구조체를 사용한다. 객체라는 개념도 비슷한 개념이다.

간단하게 말해서 하나이상의 변수들을 묶어서 개발자의 자료형으로 다시 만드는 것이라고 생각하면 편하다.

기본적인 구조는 

```c
struct		구조체 이름
{
  구조체 멤버1;
	
}					t_list;
```
